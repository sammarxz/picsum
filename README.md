# picsum
Simple image placeholder with Django and React (development)


## How to

```Bash
git clone https://github.com/sammarxz/picsum.git
cd picsum
pip install -r requirements.txt
```

- set env variables

```Bash
export SECRET_KEY='YOUR_SECRET_KEY'
export DEBUG=True
```

- Run the Project

```Bash
python manage.py runserver
```
